ASSUNTO: ÍNDECE

* OQUE É UM ÍNDICE E QUAL A SUA IMPORTÂNCIA?

Quando estamos a consultar um livro, em algumas situações queremos apenas ler um ponto específico, um determinado tema ou capítulo de modo a evitar a necessidade de ler o livro todo para chegar ao ponto que se pretende, e para resolver essa situação recorremos ao indice do livro procurando exatamente pela página do assunto que queremos consultar. 

É exatamente essa mesma ideia aplicada no cenário de banco de dados! 

Em uma base de dados o índice é um recurso usado para otimizar o retorno de dados em uma consulta, nos retornando exatamente oque foi introduzido no campo de  pesquisa sem precisar percorrer todas as linhas de um determinado campo.

QUANDO USAR O ÍNDECE?

É recomendado usar o índece Quando temos uma tabela extensa com mais de mil registros e a consulta por dados em alguns campos dessa mesma tabela é feita com muita frequência. 

Vantagens DE USAR O ÍNDICE!

Otimização de pesquisa, a consulta torna se mais rápida e dinâmica visto que os dados são buscados direito ao ponto descartando a possiblidade de percorrer todas as linhas de um campo em uma tabela de banco de dados.

NOTA: Uma tabela em banco de dados pode ter milhões de registros! Não existe um limite máximo de linhas de armazenamento de dados uma vez que os números são infinitos, a quantidade de linhas aumenta na medida que o banco vai indo crescendo.
